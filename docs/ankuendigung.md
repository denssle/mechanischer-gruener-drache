# Ankündigungspost – Mechanischer Grüner Drache

Aufgeteilt in **5 Discord-Nachrichten** (Discord-Limit: 2000 Zeichen pro Nachricht).
Kopiere jeweils den **Inhalt eines Code-Blocks** und poste ihn im Ankündigungskanal –
die `#`-Überschriften, `**Fett**`, `>`-Zitate usw. rendert Discord automatisch.
Die Blöcke nacheinander posten.

---

## Nachricht 1 – Einleitung + So nutzt man Slash-Befehle + Twitch-Warnung

```
# 🐲 Der Mechanische Grüne Drache ist da!

Ab heute läuft unser eigener Server-Bot. Er kann Twitch-Live-Meldungen, kooperatives Sport-Tracking, einen Event-Countdown, Spiel-News und mehr. Hier erfährst du **alles, was du damit machen kannst** – und wie's geht.

## ⚡ Wichtig zuerst: die alten Bots werden abgeschaltet
Damit verschwinden auch **alle bisherigen Twitch-Verknüpfungen**.
👉 **Alle müssen ihren eigenen Twitch-Kanal neu verknüpfen** (wie, steht unten), sonst kommt keine Live-Meldung mehr an. Dauert 10 Sekunden.

## 🧭 Wie benutze ich einen Slash-Befehl?
Falls du das noch nie gemacht hast – ganz easy:
> **1.** Tippe `/` ins normale Nachrichtenfeld. Discord zeigt dir eine Liste aller Befehle.
> **2.** Tippe weiter, um zu filtern (z.B. `/sport`), und klick den passenden Befehl an.
> **3.** Hat der Befehl Eingabefelder (z.B. Kilometer), erscheinen sie direkt darüber – ausfüllen, mit **Tab** zum nächsten Feld springen.
> **4.** **Enter** schickt den Befehl ab. Fertig!

Am Handy genauso: `/` tippen, aus der Liste wählen, Felder ausfüllen, senden.
```

## Nachricht 2 – Twitch

```
## 📺 Twitch – sag dem Server Bescheid, wenn du live gehst
Verknüpfe deinen Kanal einmal, und der Bot postet automatisch eine Meldung im Server, sobald du live gehst.

- `/twitch verknuepfen benutzername:<dein Twitch-Name>` → verknüpft deinen Kanal. **Das ist der Befehl, den jetzt alle neu ausführen müssen!**
- `/twitch status` → zeigt, welcher Kanal aktuell bei dir hinterlegt ist.
- `/twitch entfernen` → löst die Verknüpfung wieder.
- `/twitch hilfe` → listet alle Twitch-Befehle auf.

💡 Tipp: Willst du bei Live-Meldungen gepingt werden (oder eben nicht)? Das regelst du über die Rollen-Buttons – siehe unten.
```

## Nachricht 3 – Sport

```
## 🏃 Sport – wir sammeln gemeinsam Kilometer
Bewusst **kooperativ, kein Wettkampf**: alle zahlen auf eine gemeinsame Gesamtsumme ein. Egal ob Laufen, Radfahren, Schwimmen – jeder Kilometer zählt.

- `/sport eintragen aktivitaet:<Sportart> kilometer:<Zahl>` → trägt eine Aktivität ein.
- `/sport statistik` → deine persönliche Übersicht pro Sportart.
- `/sport gesamt` → wie weit sind wir alle zusammen schon gekommen?
- `/sport bearbeiten eintrag-id:<ID> kilometer:<Zahl>` → einen eigenen Eintrag korrigieren.
- `/sport loeschen eintrag-id:<ID>` → einen eigenen Eintrag löschen.
- `/sport hilfe` → listet alle Sport-Befehle auf.

💡 Die `eintrag-id` findest du in deiner `/sport statistik`.
```

## Nachricht 4 – Event, News, Ping-Pong

```
## 📅 Event-Countdown
- `/event countdown` → zeigt, wie lange es noch bis zum nächsten Community-Event dauert. (Den Termin legt das Team fest – du fragst einfach jederzeit „wie lange noch?".)

## 📰 Spiel-News
- `/news` → holt die **neueste News direkt aus dem Spiel** (von lotgd.de) und postet sie hier im Chat. Praktisch, ohne extra die Seite zu öffnen.

## 🏓 Ping-Pong (für zwischendurch)
- `/pingpong` → spielt eine Runde Ping-Pong mit dem Bot.
- `/pingbestenliste` → zeigt die Bestenliste.
```

## Nachricht 5 – Rollen-Buttons + Abschluss

```
## 🙋 Rollen selbst vergeben (per Klick)
An manchen Nachrichten hängen **Buttons**. Ein Klick gibt dir die passende Rolle, noch ein Klick nimmt sie wieder weg – ganz ohne Befehl. Damit kannst du z.B.:
- die **Einwohner-Rolle** holen (Regeln akzeptieren / Server freischalten),
- **Twitch-Live-Benachrichtigungen** an- oder abschalten.

Einfach auf den Button unter der jeweiligen Nachricht tippen. Deine Rückmeldung sieht nur du selbst.

**Das war's!** 🐲 Fragen, Bugs oder Wünsche? Postet sie einfach hier.
**Und nicht vergessen:** `/twitch verknuepfen` neu ausführen, sonst gibt's keine Live-Meldungen mehr. Viel Spaß!
```
